# Financial Dashboard

## 📌 Overview

This project is a financial dashboard that displays company asset data in a structured tabular format. Users can dynamically add new financial year data through a modal.

---

## ⚙️ Setup Instructions

1. Clone the repository:
   git clone https://github.com/Chopardmp/financial-dashboard.git

2. Navigate to project:
   cd financial-dashboard

3. Install dependencies:
   npm install

4. Run the project:
   npm start

---

## 🛠️ Tech Stack

- React (Parcel)
- Tailwind CSS
- React Icons

---

## ✨ Features

- Dynamic financial table rendering
- Proper asset sectioning
- Add latest year via modal
- Automatic subtotal & total calculations
- Sidebar navigation UI
- Form validation implemented

---

## 🧠 Assumptions

- Only asset fields are taken as user input
- All totals and subtotals are calculated dynamically
- No backend (frontend-only implementation)
